# Hub Service (`@apps/hub`)

itjustbong 서비스 링크를 한눈에 모아 제공하는 Vite + Vanilla TypeScript 기반 링크 허브입니다.

## 로컬 실행

```bash
pnpm dev
```

기본 포트는 `3003`입니다.

## 빌드

```bash
pnpm build
pnpm preview
```

## 배포

정적 빌드 결과물은 `dist/`에 생성됩니다.

Docker 배포는 `/Users/bong/Developer/blog.itjustbong/apps/hub/Dockerfile`을 사용합니다.
